MaveTools¶

MaveTools is a pure Python Library for bioinformatics and computational biology. It features useful tools that can be applied to MAVE data, such as that available at https://www.mavedb.org/.

Install MaveTools using pip:

pip3 install mavetools

Building a local copy of the documentation requires the following additional packages:

pip3 install sphinx
